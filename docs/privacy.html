Kaizen Scope ・ カイゼンスコープ

プライバシーポリシー

最終更新日: 2026年5月
「カイゼンスコープ」(以下「本アプリ」)は、製造業現場の作業分析を支援するスマートフォンアプリです。本プライバシーポリシーでは、本アプリが取り扱うデータについて説明します。

1. 取得する情報と利用目的

本アプリは以下の情報を端末内でのみ取り扱い、開発者および第三者のサーバーには送信しません。

情報利用目的
カメラ映像作業動画の録画 (ユーザーが「カメラで撮影」を選択した場合のみ)
マイク音声作業動画の録音 (録画機能の一部)
メディアライブラリの動画ファイルユーザーが選択した既存の動画を分析の対象として読み込むため
ユーザーが入力したテキスト (セッション名、作業名、リソース名等)端末内でのデータ管理・分析結果の表示

これらの情報は すべて端末のローカルストレージ (アプリ専用領域) に保存され、外部サーバーへの送信は行いません。

2. 第三者への提供

本アプリは、ユーザーの動画ファイルや分析データを第三者へ提供しません。

ただし、ユーザー自身が以下の操作を行った場合、その操作に応じて第三者のサービスにデータが渡る場合があります:

  • Excel/PDF レポートの共有: ユーザーが「共有」操作を行い、メールアプリやクラウドストレージ (Google Drive、OneDrive、Dropbox 等) を選択した場合、そのアプリ/サービスのプライバシーポリシーが適用されます。
  • クラッシュレポート (Sentry): アプリのクラッシュ発生時に、匿名化されたスタックトレース・端末情報・OS バージョンが Sentry 社のサーバーに送信されます。これは品質改善のみに使われ、個人を特定する情報や動画・録画データは含まれません。

3. 広告について

本アプリは無料でご利用いただけ、収益のためにモバイル広告 (Google AdMob) を表示します。AdMob は広告配信のために以下の情報を取得する場合があります:

  • 広告 ID (リセット可能な匿名 ID)
  • IP アドレス
  • 端末情報 (OS バージョン、機種等)

詳細は Google のプライバシーポリシー をご確認ください。

4. データの保存・削除

本アプリのデータは、ユーザーが下記いずれかの操作を行うまで端末に保持されます:

  • アプリ内でセッションを削除する (関連する動画ファイルも自動削除)
  • アプリ内の「全データを削除」を実行する — セッション一覧右上の ⓘ ボタン → データ管理 → 全データを削除
  • アプリをアンインストールする (すべてのデータが削除されます)

本アプリ開発者は、ユーザーのデータにアクセスする手段を持ちません。

5. お子様のプライバシー

本アプリは 13 歳未満の方を対象としていません。

6. プライバシーポリシーの変更

本ポリシーは予告なく変更される場合があります。変更後のポリシーは本ページに掲載した時点から効力を生じます。
