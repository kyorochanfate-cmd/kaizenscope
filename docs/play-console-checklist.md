## 5. アプリのリリース (最初は Internal testing 推奨)

### ステップ

1. **テスト → 内部テスト** タブ
2. **新しいリリースを作成**
3. **アプリ バンドル**: 本番 AAB をアップロード (まだないので後述)
4. **リリース ノート** (例):
   ```
   v1.0.0 初回リリース
   ・動画から要素作業を計測
   ・ガント・山積み・改善ポイント表示
   ・Excel/PDF レポート出力
   ・年間効果見込み試算
   ```
5. **テスターを追加** (まずはご自身のメール アドレスのみで OK)
6. **保存** → **レビュー** → **公開**

